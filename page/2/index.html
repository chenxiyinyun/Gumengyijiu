走四方——《梦游天姥吟留别》

发表于 2018-11-24 | 更新于 2018-12-31

作者:秦笙

   走四方，路迢迢，水长长，迷迷又茫茫，一村又一庄；看斜阳，落下去，又回来，地不老，天不荒岁月长又长。李白一生好游山玩水，寻仙问道，游遍世间美景，访遍千山万水，增长的不只是知识，还有阅历。
  梦中的天姥山也是李白向往的地方，虎鼓瑟兮鸾回车，仙之人兮列如麻，十分神奇，而转眼回望现实世界，虽比不上这云霞明灭或可睹的天姥山，却也说：大千世界，无奇不有。世界这么大，谁说不值得去看看，外面虽然有战火，有瘟疫，有疟疾，但也有佛罗伦萨的静谧，日本樱花的绚丽，阿尔卑斯的迤逦。我希望有一天我可以带上相机，来场说走就走的旅行，在世界上每个角落都留下我的身影，用相机记录世界的模样，就算没钱，也可以在那里打工攒钱，备好下一个目的地的机票，我希望有一天，我可以过上许多人想过却也只是想过的生活，既可以朝九晚五，也可以浪迹天涯。走向四方，看看世界的精彩，才是不枉走这一遭。

拾起那份珍惜——《祭十二郎文》

发表于 2018-11-24 | 更新于 2018-12-31

作者:潘虹余

   时光很短，天涯很远，这世上总有些人，非要等到千帆过尽，才开始知道回头，要等到流离失所，才开始懂得珍惜，等到物是人非，才开始怀恋。韩愈写下《祭十二郎文》为的是自己与十二郎间的骨肉亲情。苏轼曾说过:“读《十二郎文》不下泪者，其人必不友”。可见它感人至深。同时我们也从中懂得了:要珍惜眼前拥有的人和事，若待其逝去后，你留下的只有悔恨。有些人，有些事，当他们还在我们身边的时候，我们无法体会他们的重要性，当我们能体会的时候，他们可能早已走远。世上有的东西可以弥补，有些却无法弥补，唯一能做的便是珍惜。“树欲静而风不止，子欲养而亲不待”随着时光的流逝，我们在不断地长大，那么我们最应该珍惜的又何尝不是亲情?满目山河空念远，落花风雨更伤春，不如怜取眼前人，不求尽如人意，但求无悔于心。任何地方，都可以生长，任何人事，都值得珍惜，任何情感，都需回应。时光老去，依旧要拾起那份珍惜!

祭十二郎文

发表于 2018-11-24 | 更新于 2018-12-31

作者:李卓航

   言语温柔莫负，尤其等繁华落幕，谁把年华相付，终将时光倾覆。遇见你，是我的荣荣幸；离开你，是我的悲哀。韩愈和十二郎三聚三离，十二郎的逝去，令人惋惜。时光悠悠，青春渐老，白了发鬓，断了相思。过不去的终被时间带走，就这样被带走了，一切都烟消云散。可，有些记忆，注定无法抹去；就像某些人，注定无法替代。李白有句诗：长相思兮长相忆，知道相思兮无穷及。亦是如此，天不老，情难绝，心似双丝网，中有千千结。而我们，若能有幸，一起走过人生沟壑，走过四季风霜。除非黄土白骨，我守你百岁无忧。

花与水——《祭十二郎文》

发表于 2018-11-24 | 更新于 2018-12-31

作者:向苏红

  《祭十二郎文》是一篇祭文，无可厚非地要谈及生与死。
  死去的人们长眠地底，风雪刻碑，活着的人徒留思念等待成空。是谁的一句来日方长却让活着的人目睹了人走茶凉，正如你永远也不会知道，明天和意外，谁先来到。张嘉佳的《从你的全世界路过》里面说过：“阳光席卷城市，微风穿过指尘，入夜每个电台播放的情歌，沿途每条山路铺开的影子，全是你的遗嘱，而我是你唯一的遗物。”此去经年，香桃木开满坟前惹风沙，谁的思念在石碑上发芽。生与死的距离，是谁也跨不过的天堑。所以在以后空旷而寂寥的日子里，有的人选择踽踽独行，有的选择至此终年。日本德川幕府新撰组的副长土方岁三，被人称为“魔之副队”可他与局长斋藤先生师出同门，同属天然理心流，在明治维新后，斋藤先生被捕被斩首示众，从幼年相识彼此相互陪伴，三十多年的情感怎么能忘，死去的固然是勇者，可活着的人却背负着痛苦和逝者的梦想，独自走在漫漫长路上，抛弃了过去，亦看不见希望。而他一直关照过多的师弟冲田总司也身患唠咳，殁于那个八重樱漫天飘舞的五月，绚烂地似要掏空所有的生命的五月。所以土方岁三先生才会选择于次年的战争中，自杀性地冲在最前方，是因为没了斋藤和总司的余生，他也生无可恋了吧？冲田总司在病逝前留下的唯一一句俳句：“身不动，隔过黑暗，花与水。”直译为“那些隔过黑暗的花与水”。岁三先生，如果已经很累了，就请安心地合上双眼，不再扰心那些纷扰的世事，好好休息吧，那些隔过黑暗的花与水，就是远远隔了生死，也一样会花红水流，一样的美吧。也许你能在另一个世界里，穿越冗长的黑暗，寻获遗失的花与水，得到永久的宁静和解脱。

顺则生，逆则破 ——《庖丁解牛》

发表于 2018-11-24 | 更新于 2018-12-31

作者:马志强

   世间万物都有固有的规侓性，只要我们在注重的事物中多花些精力，不断地摸索，反复专研其中存在之理，久而久之，就能熟能生巧，在熟练的运用之下，把事情做得更好！庖丁有如此精湛的技艺，与他善于专研，顺乎天理是密切相关的，他从“所见无非牛”到“游刃有余”最主要的是发现了事物本质的道理。在道理上，顺从它，不仅行事时事半功倍，而且在这方面上拥有超凡的技术；在已有之理上，逆叛它，挑衅它，最终将弄得头破血流，然后才知道理不可逆。梁惠王从庖丁解牛中悟得养生之道，我们在顺乎天理中何尝不能悟得相关之理呢？顺理会获得成功，逆理将被破灭。
